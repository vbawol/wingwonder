# Wing Wonder

Website for [ventwindowprop.com](https://ventwindowprop.com)

## About

This repository hosts the website for Vent Window Prop.

## Development

This is a static website hosted on GitHub Pages. To preview locally, you can open `index.html` in your browser or use a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `main` branch.
